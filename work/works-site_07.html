[自主製作]インテリアショップ「LENTO」のECサイト

WEBサイトはこちら

架空のインテリアECサイト「LENTO」を想定し、ブランドの世界観を伝えながら商品を魅力的に見せるサイト制作を目的として制作しました。単に商品を並べるだけでなく、特集ページや商品ページを通して「インテリアを楽しむ暮らし」を提案する構成を意識しています。また、ECサイトとしての基本的なUI構造（商品一覧・商品詳細・特集ページなど）を実装し、実際のECサイトを想定したページ構成で制作しました。

URL

lento/index.html

サイトの概要

商品紹介と特集コンテンツを通して、インテリアの楽しみ方を提案及びオンラインショップからの購入者数の増加

ターゲット

都市で暮らす30代以上のインテリア好きの男女をメインターゲットに設定しています。
シンプルで落ち着いたインテリアや、日常を少し豊かにしてくれる雑貨や家具を探しているユーザーを想定し、ゆったりと商品を選べるようなサイト体験を目指しました。

また、インテリア初心者でも商品イメージが湧きやすいよう、特集ページではコーディネート提案型のコンテンツ構成にしています。

デザインについて

「静かで穏やかな暮らし」をテーマに、余白を活かしたシンプルで落ち着いたデザインを意識しました。
写真を主役にしたレイアウトにすることで、インテリア商品の質感や雰囲気が伝わる構成にしています。
また、ECサイトでは商品が見やすいことが重要なため、文字サイズ・余白・レイアウトのバランスを調整し、読みやすく整理されたUIになるよう設計しました。

コーディングについて

HTML / CSS / JavaScript を使用してコーディングしています。レスポンシブ対応を行い、PC・スマートフォンのどちらでも快適に閲覧できるように設計しました。また、ECサイトを想定し商品カードや商品一覧などは再利用しやすい構造でマークアップしています。トップページではメインビジュアルにスライダーを実装し、サイトの世界観を印象的に見せる構成にしています。

担当

コンセプト立案（3時間）、ペルソナ設定（2時間）、サイトマップ制作（1時間）、ワイヤーフレーム制作（20時間）、商品写真選定及び加工（10時間）、デザイン・コーディング（30時間）、ロゴ、ファビコン制作（1時間）

使用ソフト

Photoshop、Illustrator、Visual Studio Code

特集ページについて

本サイトでは、商品の魅力をより伝えるために特集ページを制作しました。単に商品を一覧で紹介するのではなく、インテリアのコーディネートをイメージできる構成を意識しています。
ラグ・クッション・ブランケット・ポスターといったアイテムをテーマごとに紹介し、空間全体の雰囲気が伝わるよう写真とテキストのバランスを調整しました。
また、スクロールに合わせて自然にコンテンツが展開されるよう、大きなビジュアルと余白を活かしたレイアウトを採用し、 読みやすくブランドの世界観が伝わるページ構成にしています。
