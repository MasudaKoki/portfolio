SKILL

HTML

既存サイトのHTML構造を理解した上で、企業サイトや製品ページの更新業務を実務で担当しています。運用フェーズを前提に、既存ルールやSEOを意識したマークアップに対応可能です。

CSS3

レスポンシブ対応された既存サイトの修正・調整を実務で行っています。レイアウト崩れや表示差異の調整など、運用に必要な細かな対応が可能です。

SCSS

既存のSCSSファイルを理解し、変数やネスト構造を踏まえた軽微な修正を行った経験があります。既存設計を崩さずに調整することを意識しています。

jQuery

既存サイトで使用されているjQueryコードを読み取り、簡単な修正やプラグイン調整を行った経験があります。既存挙動を保ちながらの対応を得意としています。

JavaScript

既存コードの理解を中心に使用しており、業務で必要な範囲については調べながら着実に対応しています。軽微な実装調整や不具合修正にも取り組んでいます。

Photoshop

Webサイト・ECサイト運用において、画像の切り抜き、リサイズ、書き出し、簡易バナー制作などを実務で行っています。スピード感を持った素材対応が可能です。

Illustrator

ECサイトのリニューアル時にロゴ検討案の作成で使用しました。基本的な図形作成や文字組みが可能で、紙・Web問わずビジュアル制作の基礎があります。

WordPress

WordPressを使用したページ更新やコンテンツ修正を実務で担当しています。固定ページの編集や既存内容の差し替えなど、運用業務に対応可能です。

Movable Type

Movable Typeを使用したページ更新業務の経験があります。既存コンテンツの修正・更新作業を中心に、ルールに沿った運用対応を行ってきました。

ec-force

ec-forceを使用したECサイトの商品登録や商品情報の修正を実務で行っています。細かな確認作業を含め、正確性を意識して対応しています。

EC-CUBE

EC-CUBEを使用したECサイトの商品登録や情報更新を実務で担当しています。日々の運用更新に必要な基本操作に対応可能です。

AEM

Adobe Experience Manager（AEM）を使用したページ更新を実務で経験しています。管理画面からの更新作業を中心に、既存サイトの運用に携わっていました。

Figma / XD

職業訓練校の課題制作にてデザインカンプ作成で使用しました。簡単な画面設計や、画面遷移確認のためのプロトタイプ作成が可能です。

Git / GitHub

個人制作および業務内で、コミットや更新管理などの基本操作を行っています。既存フローに合わせながら、丁寧な更新を意識しています。

ABOUT

職業訓練校修了後、Web制作会社および事業会社にて、企業サイトやECサイトの更新・運用業務に携わってきました。ページ更新や画像編集、バナー制作、特集ページの作成など、日々の運用実務を担当しています。

業務の中で意識しているのは、単に更新するだけでなく「情報や商品の魅力が伝わる状態に整えること」です。レイアウトや見せ方を意識しながら、ユーザーにとって分かりやすく、興味を持ってもらえる表現を心がけています。

インテリアや家具に興味があり、プロダクトの魅力や世界観をWeb上で伝える仕事に携わりたいと考えています。今後は、運用業務に加えて企画や改善にも積極的に取り組みながら、より価値のあるコンテンツを生み出していきたいと考えています。
